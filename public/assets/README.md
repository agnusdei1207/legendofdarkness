# 에셋 디렉토리

이 폴더에는 게임 에셋(이미지, 사운드 등)을 저장합니다.

## 구조

```
assets/
├── items/          # 아이템 스프라이트
├── monsters/       # 몬스터 스프라이트
├── characters/     # 캐릭터 스프라이트
├── tiles/          # 타일셋
├── ui/             # UI 요소
└── sounds/         # 사운드 이펙트
```

## 픽셀 아트 가이드라인

- 크기: 16x16, 32x32 픽셀
- 포맷: PNG (투명 배경)
- 색상: 제한된 팔레트 사용 (8-16 색상)
- 도구: Aseprite, Piskel, GraphicsGale
